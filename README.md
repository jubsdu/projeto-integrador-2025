Desenvolver um **sistema web integrado**, com interface interativa e conexão a um **banco de dados relacional**, permitindo a troca de informações entre o front-end e o back-end.

# Tecnologias utilizadas

| **Back-end** | Python(Flask)
| **Banco de Dados** | MySQL e SQL Alchemy
| **Versionamento** | Github
